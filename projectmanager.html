Technical Project Management

Project Management is the application of techniques and experience to achieve specific goals within an agreed timeframe and budget. Technical project management typically applies these disciplines to IT projects.

With experience of both agile and waterfall methodologies, we can provide Technical PM services to your organisation.Some examples of Mike's previous projects are:

  • Software development - Managed the development of a number of software applications for an independent software consultancy
  • Platform migration - Mananged a WMS platform migration for a large Logistics company.
  • Database upgrade - Managed the database upgrade for a leading General Insurance company.
  • Document Management - Managed the implementation of an online document management system for that same General Insurance company.
